# Aarogya Mitra - AI Health Coach 🏥🤖

Aarogya Mitra is an AI based health assistant designed to help users with basic health guidance, symptom checking, and wellness advice.

---

## 🚀 Features

- AI based health suggestions
- Symptom checking
- Basic healthcare guidance
- User friendly interface

---

## 🛠 Technologies Used

- Python
- AI / Machine Learning
- APIs
- HTML
- CSS

---

## ⚙ Installation

1. Clone the repository

git clone https://github.com/Amritesh373/-Aarogya-mitra-AI-health-coach.git

2. Go to the project folder

cd -Aarogya-mitra-AI-health-coach

3. Install requirements

pip install -r requirements.txt

4. Run the program

python main.py
